# Drone Delivery Service Benchmark

A comprehensive benchmarking framework for drone delivery services with an implementation of the exact algorithm presented in the literature for drone allocation, particularly focusing on the **Drone Routing with Energy Function: Formulation and Exact Algorithm** paper.

## Overview

This repository implements a benchmarking system for evaluating drone allocation strategies in delivery networks. The project features:

- **Exact Algorithm Implementation**: Based on the renowned research paper "Drone Routing with Energy Function: Formulation and Exact Algorithm"
- **Benchmark Generation**: Systematic test case generation following the methodology described in Appendix B of the reference paper
- **Streamlit Application**: Interactive web interface for exploring and analyzing benchmark results
- **Comprehensive Analysis**: Tools for evaluating algorithm performance and drone allocation efficiency
